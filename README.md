# lease-finance-app

## Getting Started

### Clone repository

```sh
git clone git@github.com:pankaj3399/lease-finance-app.git

cd lease-finance-app
```

### Install pnpm

```sh
npm i --global --save-dev pnpm@latest
```

### Start client app

```sh
cd client
pnpm install
pnpm run dev
```

### Start server app

```sh
cd apps/backend
pnpm install
pnpm run dev
```
